# Wikipedia Dynamic Website

This is a simple dynamic website inspired by Wikipedia, built using **HTML, CSS, and JavaScript**.

The project shows different sections on the same page without reloading. Users can click on books to read more details and go back to the home page easily.

---

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap

---

## What this project does
- Displays book information
- Changes sections dynamically using JavaScript
- Works well on both mobile and laptop screens
- Uses clean and simple UI
